si votre prestashop n'est pas directement a la racine de votre serveur,
vous devez copier les fichiers du dossier "fichiers_a_copier" dans votre emplacement theme/votre theme manuellement.

1 fichier address.tpl
et un dossier /js contenant 4 fichier javascript
geo.js, jquery.ui.addresspicker.js, jquery-1.4.4.min.js, jquery-ui-1.8.7.min.js

Copier manuellement les ficiers dans votre dossier
/themes/ "votre theme" / adress.tpl /themes/ "votre theme" / js / (...)


Compatible et teste avec Prestashop 1.6.x
